### Step 2: Get SHA-1 Fingerprint (Optional but Recommended)

Run this command to get your debug key SHA-1:

```bash
# macOS/Linux
keytool -list -v -keystore ~/.android/debug.keystore -alias androiddebugkey -storepass android -keypass android | grep SHA1

# Or use flutter for easier method
flutter doctor -v | grep -A 10 "Android Studio"
```

Then update the SHA-1 in Firebase Console:
1. Project Settings → Select your Android app
2. Scroll to "SHA certificate fingerprints"
3. Click "Add fingerprint"
4. Paste your SHA-1

## Troubleshooting

### Issue: "google-services.json not found"
Make sure files are in exact paths:
```bash
ls android/app/src/user/google-services.json
ls android/app/src/provider/google-services.json
```

### Issue: Package name mismatch error
Ensure your Firebase Console app registration exactly matches:
- User: `com.daem.diyafatukum`
- Provider: `com.daem.diyafatukum.provider`

### Issue: Firebase not initializing
Update `firebase_options.dart` with values from `google-services.json`:
- `apiKey`
- `appId`
- `messagingSenderId`
- `projectId`
- `storageBucket`

### Issue: Build fails with "permission denied"
Make scripts executable:
```bash
chmod +x *.sh
```
